# AWS Certified Cloud Practitioner (CLF-C02) Resources

## Knowledge

### Fontes oficiais AWS (primárias — sempre preferir)
- [Exam Guide oficial CLF-C02 (HTML)](https://docs.aws.amazon.com/aws-certification/latest/examguides/cloud-practitioner-02.html)
  Fonte da verdade sobre domínios, pesos e o que é cobrado. Use para: validar escopo de qualquer lição.
- [Exam Guide oficial CLF-C02 (PDF)](https://docs.aws.amazon.com/pdfs/aws-certification/latest/cloud-practitioner-02/cloud-practitioner-02.pdf)
  Versão para imprimir/estudar offline. Use para: revisar a lista completa de tarefas por domínio.
- [Página da certificação (logística do exame)](https://aws.amazon.com/certification/certified-cloud-practitioner/)
  65 questões, 90 min, US$100, validade 3 anos. Use para: detalhes de inscrição e formato.
- [Overview of Amazon Web Services (whitepaper)](https://docs.aws.amazon.com/whitepapers/latest/aws-overview/introduction.html)
  Panorama de todos os serviços por categoria. Use para: Domínio 3 (serviços) e visão geral.
- [Six advantages of cloud computing](https://docs.aws.amazon.com/whitepapers/latest/aws-overview/six-advantages-of-cloud-computing.html)
  As 6 vantagens canônicas (muito cobradas). Use para: Domínio 1 — proposta de valor da nuvem.
- [What is Cloud Computing?](https://aws.amazon.com/what-is-cloud-computing/)
  Definição e tipos de nuvem. Use para: Domínio 1 — fundamentos.
- [How AWS Pricing Works (whitepaper)](https://docs.aws.amazon.com/whitepapers/latest/how-aws-pricing-works/welcome.html)
  Princípios de preço. Use para: Domínio 4 — billing e pricing.
- [Shared Responsibility Model](https://aws.amazon.com/compliance/shared-responsibility-model/)
  Modelo de responsabilidade compartilhada. Use para: Domínio 2 — segurança (muito cobrado).
- [AWS Well-Architected Framework](https://aws.amazon.com/architecture/well-architected/)
  Os pilares de boa arquitetura. Use para: Domínio 1/3 — boas práticas.

### Treinamento gratuito
- [AWS Skill Builder — Exam Prep CLF-C02 (gratuito)](https://explore.skillbuilder.aws/learn)
  Curso oficial de preparação + question sets. Use para: estudo guiado e simulados oficiais.
- [AWS Cloud Practitioner Essentials (curso digital gratuito)](https://aws.amazon.com/training/digital/aws-cloud-practitioner-essentials/)
  Curso introdutório oficial. Use para: base completa dos 4 domínios.

## Wisdom (Communities)
- [r/AWSCertifications](https://www.reddit.com/r/AWSCertifications/)
  Subreddit muito ativo com relatos de prova, dicas e dúvidas. Use para: o que realmente cai, calibrar
  expectativa de dificuldade, escolher material de simulado. (Confirmar com o aluno se quer participar.)
- [AWS re:Post](https://repost.aws/)
  Q&A oficial da AWS. Use para: dúvidas técnicas conceituais com respostas confiáveis.
