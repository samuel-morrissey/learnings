# Régua de autonomia (chamada única / workflow / agente) + action loop

O aluno demonstrou entender, com suas próprias palavras, que chamada única, workflow e agente se distinguem por **quem controla o caminho de execução** (você → código → modelo), e definiu o action loop corretamente como "age → recebe feedback da ação → decide a próxima ação". Isto estabelece o piso do Domínio 1 (27%) e habilita ensinar os 5 padrões de workflow em seguida.

## Evidence
Pseudo-cola enviada no chat (sessão 1, 2026-06-19): definiu os quatro conceitos corretamente. Dois refinamentos aplicados: (1) workflow orquestra LLMs+tools+código em caminho fixo, não só "chamadas únicas"; (2) "conversacional" é formato de interação, não posição na régua — armadilha de prova ainda não testada em recall.
